# Homebrew Tap

## Available Formulas

| Formula | Description |
|---------|-------------|
| [`ask`](https://github.com/dl-alexandre/App-StoreKit-CLI) | CLI for the App Store Server API |
| [`ams`](https://github.com/dl-alexandre/Apple-Map-Server-CLI) | Apple Map Server CLI |
| [`aca`](https://github.com/dl-alexandre/Advance-Commerce-CLI) | Advanced Commerce API CLI |
| [`gdrv`](https://github.com/dl-alexandre/Google-Drive-CLI) | Google Drive CLI with full read/write support |
| [`gpd`](https://github.com/dl-alexandre/Google-Play-Developer-CLI) | Google Play Developer CLI for managing Play Console operations |

## Install

```bash
brew tap dl-alexandre/tap
brew install ask
brew install ams
brew install aca
brew install gdrv
brew install gpd
```
